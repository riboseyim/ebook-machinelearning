# 基于OpenCV实现美女打分机

## Enviroment

```bash
brew install opencv3
# dependencies for opencv: eigen, lame, x264, xvid, ffmpeg, libpng,
# libtiff, ilmbase, openexr, gdbm, python, xz, python3, numpy, tbb

npm install --save opencv4nodejs
```
